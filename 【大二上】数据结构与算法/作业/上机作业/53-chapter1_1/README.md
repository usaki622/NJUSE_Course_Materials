找出从自然数 1, 2, …, n 中任取r个数的所有组合, 编一个递归算法.

你可以按 **任何顺序** 返回答案。

提示：

1 <= n <= 20

0 <= r <= n
 
例子: n = 5, r = 3 

|      |      |      |
| ---- | ---- | ---- |
|  5    |    4  |  3    |
|   5   |    4  |   2   |
|  5    |    4  |   1   |
|5| 3| 2|
|5 |3 |1|
|5 |2 |1|
|4 |3 |2|
|4 |3 |1|
|4 |2 |1|
|3 |2 |1|
